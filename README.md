Website for biz-icon
===============

[![Build Status][travis-image]][travis-url]

Usage
-----
Run these commands:

    $ git clone git://github.com/rickyleung/biz-icon-web.git
    $ cd biz-icon-web
    $ npm install
    $ npm start

Now you should have a biz-icon-web server running on http://localhost:3000.
